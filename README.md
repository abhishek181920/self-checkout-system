# SCANEZY Self-Checkout System

A smart self-checkout system for retail stores with mobile-first design.

## Features

- Mobile-first Progressive Web App (PWA)
- OTP-based authentication
- Barcode scanning for product identification
- Shopping cart management
- UPI payment integration
- Store management dashboard

## Deployment

This application is ready for deployment to Vercel via GitHub integration.

## APK Generation

After deployment, you can generate an Android APK using PWABuilder:
1. Visit [PWABuilder](https://www.pwabuilder.com)
2. Enter your deployed URL
3. Follow the steps to generate Android APK
4. Download and install on your device

## Technologies Used

- HTML5, CSS3, JavaScript (React)
- Progressive Web App (PWA) features
- Service Worker for offline support
- Web Manifest for installability

## Setup Instructions

1. Clone this repository
2. Deploy to Vercel
3. Access the application via the provided URL
